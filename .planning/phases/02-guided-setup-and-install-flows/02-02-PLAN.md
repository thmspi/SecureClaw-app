Create zustand store following research Pattern 1:

1. Create `src/renderer/stores/` directory if needed

2. Create `src/renderer/stores/wizard-store.ts`:
   - Import { create } from 'zustand'
   - Import { persist, createJSONStorage } from 'zustand/middleware'
   
   Define types inline (will be moved to shared when contracts exist):
   ```typescript
   type WizardStep = 'welcome' | 'prerequisites' | 'install' | 'complete';
   type PrerequisiteStatus = 'passed' | 'failed' | 'warning' | 'pending';
   
   interface PrerequisiteCheck {
     id: string;
     name: string;
     description: string;
     status: PrerequisiteStatus;
     result: { value?: string; required?: string; message: string };
   }
   
   interface InstallProgressState {
     status: 'idle' | 'running' | 'paused' | 'failed' | 'completed';
     currentStep: number;
     totalSteps: number;
     stepName: string;
     progress: number;
     estimatedTimeRemaining?: number;
     error?: { message: string; details?: string };
     logs: string[];
   }
   ```
   
   Interface `WizardState`:
   - currentStep: WizardStep
   - completedSteps: WizardStep[]
   - prerequisites: Record<string, PrerequisiteCheck>
   - prerequisitesLoading: boolean
   - allPrerequisitesPassed: boolean
   - install: InstallProgressState
   - selectedTarget: 'openclaw' | 'nemoclaw' | null
   
   Actions:
   - setStep(step: WizardStep): Updates currentStep, adds previous to completedSteps
   - canNavigateTo(step: WizardStep): Returns true if step is current or in completedSteps (D-02)
   - setPrerequisitesLoading(loading: boolean)
   - updatePrerequisite(id: string, check: PrerequisiteCheck)
   - setAllPrerequisitesPassed(passed: boolean)
   - updateInstallProgress(progress: Partial<InstallProgressState>)
   - appendLog(line: string)
   - setSelectedTarget(target: 'openclaw' | 'nemoclaw')
   - reset(): Clear all state to initial values
   
   Create store with persist middleware:
   - name: 'secureclaw-wizard'
   - storage: createJSONStorage(() => localStorage)
   - partialize: Only persist currentStep, completedSteps, prerequisites (not logs or running state)
   
   Export useWizardStore hook
  </action>
