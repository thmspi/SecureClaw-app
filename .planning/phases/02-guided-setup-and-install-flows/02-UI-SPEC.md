### Recovery Components
- **Dialog** — Cancel/rollback confirmation dialogs
- **Alert** — Error display with expandable technical details

---

## Interaction Contracts

### Wizard Navigation (from D-02, D-04)
- Step indicator at top: `1 > 2 > 3 > 4`
- Clickable back-jumps to completed steps only
- Forward navigation blocked until current step complete
- Animated transitions between steps (from D-03)

### Prerequisite Checks (from D-05 through D-08)
- Automatic checks on step entry
- Manual "Re-check" button available
- Install button disabled until all checks pass
- Per-item status: green checkmark / red X with fix instructions

### Install Progress (from D-09 through D-12)
- Named steps with overall progress: "Installing OpenClaw... Step 1 of 5", then "Installing NemoClaw... Step 3 of 5"
- Raw log hidden by default, "Show details" to expand
- Estimated time remaining when determinable
- Inline error with plain-language explanation

### Recovery Behavior (from D-13 through D-16)
- Retry resumes from failed step (completed steps skipped)
- Cancel shows confirmation dialog before cleanup
- Full rollback to pre-install state on cancel/failure
- Install state tracked in local DB (survives app restart)
